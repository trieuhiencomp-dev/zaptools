üîí Privacy & Security
Is my data safe when using ZapTools? ‚ñº

Absolutely! Most of our tools process files directly in your browser (client-side processing), meaning your files never leave your device. For tools that require server processing, we process your files temporarily and delete them immediately after. We never store, access, or share your files.

Do you store my uploaded files? ‚ñº

No, we do not store your files. For client-side tools, files are processed entirely in your browser. For server-side tools, files are processed in memory and deleted immediately after processing completes.

Can I use ZapTools for sensitive documents? ‚ñº

While we take privacy seriously and process most files client-side, we recommend using caution with extremely sensitive documents. For maximum security with sensitive files, consider using our client-side tools which process everything locally in your browser.

Do you track my activity? ‚ñº

We use Google Analytics to understand how visitors use our site (page views, popular tools, etc.) but we do not track or store information about the files you process. See our Privacy Policy for complete details.

Are there file size limits? ‚ñº

For client-side tools, file size is limited only by your device's memory. For server-side tools, we typically support files up to 50-100MB depending on the tool. Larger file support may be added based on user feedback.

Can I use ZapTools offline? ‚ñº

Currently, ZapTools requires an internet connection to load the tools. However, once loaded, many client-side tools can work without an active connection since they process files locally in your browser.

How many files can I process at once? ‚ñº

This depends on the specific tool. Many tools support batch processing of multiple files. There's no daily limit on the number of files you can process.

Why is my tool taking a long time to process? ‚ñº

Processing time depends on several factors:

  • File size (larger files take longer)
  • Your device's processing power
  • Internet connection speed (for server-side tools)
  • Complexity of the operation

Client-side tools are generally faster since they process locally on your device.

What file formats are supported? ‚ñº

Supported formats vary by tool. Common formats include:

  • Images: JPG, PNG, GIF, WebP, BMP, SVG
  • Documents: PDF, DOCX, TXT
  • Data: JSON, XML, CSV, YAML
  • Code: JavaScript, HTML, CSS, SQL

Each tool specifies supported formats on its page.
