# Hướng dẫn cải thiện Google Indexing cho ZapTools

## 1. Submit lên Google Search Console
- URL: https://search.google.com/search-console
- Add property: https://zaptools.org
- Submit sitemap: https://zaptools.org/sitemap.xml
- Request indexing cho Index.html

## 2. Submit lên Bing Webmaster Tools
- URL: https://www.bing.com/webmasters
- Add site: https://zaptools.org
- Submit sitemap: https://zaptools.org/sitemap.xml

## 3. Tăng tốc độ index
- ✅ Sitemap.xml đã tạo
- ✅ Robots.txt đã tạo
- ✅ Google Analytics tracking đã có
- ⏳ Chờ Google crawl (2-7 ngày)

## 4. Tạo backlinks
- Share link trên social media
- Submit lên directories:
  - https://www.producthunt.com
  - https://www.reddit.com/r/SideProject
  - https://news.ycombinator.com

## 5. Kiểm tra sau 3-7 ngày
- Vào Google Search Console
- Xem "Coverage" report
- Kiểm tra số trang đã index

## 6. Nếu vẫn không index sau 2 tuần
- Request indexing cho từng trang quan trọng
- Kiểm tra Google Search Console errors
- Đảm bảo website không bị chặn bởi robots.txt

## Tips:
- Google ưu tiên index trang có content chất lượng
- Trang có backlinks được index nhanh hơn
- Update content thường xuyên để Google quay lại crawl
